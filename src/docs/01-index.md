## Get Start

```text
git clone https://github.com/BoxPistols/fractal-styleguide.git
```

### first setting \(once\)

```text
yarn run install
```

### start move

```text
yarn run start
```

### sass compile

* sass to css

```text
yarn run css
```

### build

* static files
* compile to docs/

```text
yarn run build
```
