## Get Start

```git
git clone https://github.com/BoxPistols/fractal-styleguide.git
```

### first setting (once)

```
yarn run install
```

### start move

```
yarn run start
```

### sass compile
- sass to css

```
yarn run css
```



### build
- static files
- compile to docs/

```
yarn run build
```
